To create your own glyphs for toggles, simply provide a .png file with the following naming convention: <toggle>-<state>(.png|@2x.png|@3x.png). For example, a set of glyphs for the wifi toggle would be:

wifi-enabled@2x.png
wifi-disabled@2x.png

Glyphs may be up to 47x47px in size (94x94px for @2x, 141x141px for @3x).
